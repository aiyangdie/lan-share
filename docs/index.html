快速开始

  1. 电脑：下载 Windows 版 → 解压 → 双击 LanShare.exe
  2. 手机：下载 Android 版 → 安装 → 点「附近电脑」连接
  3. 传文件：手机上传 → 电脑 uploads/；电脑放入 shared/ → 手机下载

连接不上？

  • 电脑已运行 LanShare，手机和电脑同一 WiFi
  • 防火墙允许 8787 端口，关闭 VPN 后重试
