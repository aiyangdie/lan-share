LanShare · 电脑互传

手机和电脑在同一 WiFi 下互传文件。无需云盘、无需账号，数据不出局域网。

v1.2.0 · 支持自动发现附近电脑

国内访问 GitHub 可能超时。本页下载已改用官网直链 + 国内镜像，无需打开 github.com。

自动发现

手机安装 APK 后自动搜索局域网内的电脑，无需手填 IP。

一键启动

Windows 解压后双击 LanShare.exe，自动打开控制台。

双向使用

电脑运行服务，手机即可连接；同一 WiFi 即可互传。

快速开始

  1. 电脑：下载 Windows 包 → 解压 → 双击 LanShare.exe
  2. 手机：安装 APK → 打开 → 点「附近电脑」自动发现
  3. 传文件：手机上传进电脑 uploads/；电脑文件放进 shared/ 供手机下载

连接不上？

  • 确认电脑已运行 LanShare（窗口不要关）
  • 手机和电脑连接同一 WiFi
  • Windows 防火墙选择「允许访问」
  • 关闭 VPN 后重试，或手动输入 电脑IP:8787
