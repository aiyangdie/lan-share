LanShare · 电脑互传

手机和电脑在同一 WiFi 下互传文件。无需云盘、无需账号、数据不出局域网。v1.2 支持自动发现附近电脑。

下载最新版 GitHub 开源

自动发现

手机安装 APK 后自动搜索局域网内的电脑，无需手填 IP。

一键启动

Windows 解压后双击 LanShare.exe，自动打开网页控制台。

双向使用

电脑装了服务，手机就能连；手机装了 APK，电脑浏览器也能打开同一界面。

快速开始

  1. 电脑：下载 Windows 便携包，解压，双击 LanShare.exe
  2. 手机：安装 Android APK，打开后点「附近电脑」自动发现
  3. 传文件：手机上传 → 电脑 uploads/；电脑放入 shared/ → 手机下载

下载

Windows 电脑端lan-share-v1.2.0-windows-portable.zip · 含 LanShare.exe → Android 手机端lan-share-v1.2.0-android.apk →

连接不上？

  • 确认电脑已运行 LanShare（任务栏/命令行窗口不要关）
  • 手机和电脑连接同一 WiFi（不要用访客网络）
  • Windows 防火墙弹出提示时选择「允许访问」
  • 关闭 VPN / 代理后重试
  • 仍不行可手动输入 电脑IP:8787
