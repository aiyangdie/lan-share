LanShare · 电脑互传

手机和电脑在同一 WiFi 下互传文件。数据不出局域网，无需云盘、无需注册账号。

MIT 开源 v1.2 自动发现 Windows / Android / iPhone Node.js ≥ 18

核心特性

轻量、本地、可自部署。PC 当文件网关，手机用同一套 Web UI（浏览器 / APK / PWA）。

📡

UDP 自动发现

手机搜索附近电脑，无需手输 IP。v1.2 起支持三层发现策略。

⚡

Windows 双击 LanShare.exe，自动打开 :8787 并创建数据目录。

系统架构

PC 运行 Node 服务作为中枢；手机通过 HTTP 调 API，Android 额外用原生 UDP 加速发现。

自动发现流程

PC 广播
type:lanshare
→
UDP :38787
→
手机监听 / 扫描
→
GET /api/health
→
显示附近电脑

备用策略：服务端 /api/discover/scan 或客户端扫描 192.168.x.1~254（批量探测，超时 900ms）。

技术细节

开源仓库：github.com/aiyangdie/lan-share

技术栈

  • 服务端：Node.js ≥ 18 · ES Module · server.mjs
  • 上传解析：Busboy（multipart）
  • 客户端：原生 HTML/CSS/JS（mobile-app/）
  • Android：WebView + Java 桥接（DiscoveryBridge）
  • Windows：C# 启动器 + 内嵌 Node 便携包

关键常量

  • HTTP 端口：8787（环境变量 PORT 可覆盖）
  • UDP 发现：38787
  • 协议标识：lanshare
  • 服务校验：/api/health 返回 name: lan-share
  • Peer 缓存 TTL：30 秒

HTTP API

方法路径说明
GET/api/health健康检查，返回版本、IP、端口
GET/api/discover/peersUDP 被动发现的附近电脑列表
GET/api/discover/scan主动扫描当前 /24 子网
GET/api/roots根目录：uploads / shared
GET/api/browse/{root}/{path}浏览文件夹
GET/api/download/{root}?p=...下载文件（支持 Range 断点）
POST/api/upload?target=&path=multipart 上传文件

快速开始

1
电脑启动服务

下载 Windows 版 → 解压 → 双击 LanShare.exe。浏览器会自动打开 http://127.0.0.1:8787/，同目录生成 shared/ 与 uploads/。

2
手机连接

安装 Android APK，确保与电脑同一 WiFi。打开 App →「附近电脑」自动发现 → 点击连接。iPhone 用 Safari 访问电脑 IP:8787。

3
互传文件

手机「上传」→ 文件进电脑 uploads/；电脑文件放入 shared/ → 手机「电脑共享」里下载。

开发者

git clone https://github.com/aiyangdie/lan-share.git
cd lan-share && npm install && npm start   # localhost:8787
npm run sync                              # 同步 UI 到 public / Android
npm run build:release                     # 打包全部安装包

常见问题

手机找不到附近电脑？

确认电脑已运行 LanShare、手机与电脑在同一 WiFi（非访客网络）。关闭 VPN。Windows 防火墙放行 8787 端口。仍不行则在手机端手动输入 http://电脑IP:8787。

连上但上传失败？

检查磁盘空间与文件夹权限。服务端 uploads/ 需可写。文件名含中文时服务端已做 UTF-8 乱码修复（fixMojibake）。

iPhone 怎么用？

Safari 打开 http://电脑局域网IP:8787，点击分享 →「添加到主屏幕」。即为 PWA，体验接近 App（iOS 无原生 UDP 发现，建议手动输入 IP 或扫电脑屏幕上的地址）。

数据安全吗？

文件仅在局域网内传输，不经 LanShare 官方服务器。建议在可信 WiFi 下使用；如需加密可考虑 VPN 或 HTTPS 反向代理（默认 HTTP 明文，适合内网）。

和微信 / 网盘比有什么优势？

无大小限制（受磁盘与网络影响）、无账号、速度快（局域网直连）、可批量传文件夹。适合同一办公室 / 家庭内设备互传。
